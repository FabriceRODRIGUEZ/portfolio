Races de chats

Informations générales :

  • Type de projet : Application mobile
  • Langage utilisé : Dart, SQL
  • Technologies utilisées : Flutter, Sqflite
  • Période de développement : 09/2024
  • Contexte : Bachelor développement - Cours Flutter
  • Statut : Terminé

Description du projet :

     Cette application récupère une liste de races de chats et des données sur chacune d'entre elles sur une API REST (The Cat API), puis permet à l'utilisateur de visionner les informations (Capture 1 et Capture 2) et d'enregistrer des races dans une liste de favoris (Capture 3) stockée localement dans une base de données SQLite. Le module Flutter permet de faire de la programmation par composants pour la création des pages et le système de favoris fonctionne selon le design pattern Producer/Consumer.
