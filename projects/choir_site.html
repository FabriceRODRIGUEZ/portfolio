Site de chorale

Informations générales :

  • Type de projet : Site web
  • Langage de programmation : PHP, JavaScript, HTML, CSS
  • Technologies utilisées : Symfony
  • Période de développement : Depuis 02/2025
  • Contexte : Activité associative
  • Statut : En cours

Description du projet :

Il s'agit d'un site vitrine pour l'association de chorale "Praise Gospel" comportant un espace privé pour les membres de l'association, afin d'y retrouver les paroles et les enregistrements des chants travaillés. L'authentification est faite par cookies de session à l'aide d'un serveur Symfony. L'ensemble du site est doté d'un design responsive de manière à s'adapter aux appareils mobiles.

La partie publique du site est constituée d'une page d'accueil (Captures 1 et 2), d'une page de présentation de la chorale (Captures 3 et 4), d'une page présentant les membres avec un trombinoscope (à venir), d'une page montrant les anciens concerts (Captures 5 et 6), d'une page de contacts (Captures 7 et 8), et enfin d'une page de connexion à l'espace privé (Captures 9 et 10).

Lien vers le site : https://praisegospel.fr
