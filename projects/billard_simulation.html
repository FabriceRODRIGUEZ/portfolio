Simulation de billard

Informations générales :

  • Type de projet : Logiciel desktop
  • Langage utilisé : Python
  • Technologie utilisée : Tkinter
  • Période de développement : 12/2021
  • Contexte : Licence 2 Physique - UE Maths
  • Statut : Inachevé

Description du projet :

     Ce logiciel à pour objectif de simuler un jeu de billard pool (blackball) en appliquant la mécanique des solides afin de calculer les effets des chocs élastiques et inélastiques. Les éléments à modéliser sont les chocs entre les billes, les chocs entre les billes et les bandes, les frottements de l'air sur les billes et les frottements du tapis sur les billes.

     Cependant, le temps de développement beaucoup trop court pour un projet aussi ambitieux ne m'a pas permis de le terminer. Les seuls éléments réellement développés sont le déplacement de la bille blanche seule avec les rebonds sur les bandes (Capture 1), ainsi que la situation initiale avec les billes de couleur (Capture 2). Les paramètres définissables par l'utilisateur pour le déplacement de la bille blanche sont la direction et la vitesse initiales (Vidéos).
