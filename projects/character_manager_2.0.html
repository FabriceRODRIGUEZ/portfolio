Gestionnaire de personnages 2.0

Informations générales :

  • Type de projet : Application web
  • Langages utilisés : Javascript, SQL, HTML, CSS
  • Technologies utilisées : Node.js, Docker, Figma
  • Période de développement : 03/2025 - 06/2025
  • Contexte : Bachelor Développement Web et Mobile
  • Statut : Terminé

Description du projet :

     Cette application de gestion de personnages est conçue selon une architecture MVC (Modèle Vue Contrôleur), avec l'utilisation de Node.js pour le serveur, notamment les modules Express.js pour les routes et le module Pg pour les requêtes SQL. Les données sont stockées dans une base de données relationnelle PostgreSQL.
